# angular-es6-example
* angular 1.x
* es6
* webpack
* bootstrap
* oclazyload
* angular-ui-router

## Install
Make sure have install karma-cli
`npm install -g karma-cli`
Make sure have install protractor
`npm install -g protractor`
`npm install`
## Development
`npm run dev`
http://localhost:8080
## Test(Karma Jasmine Protractor)
unit test 
`npm run test`
e2e test 
`npm run e2e`


## Deploy
`npm run build`
